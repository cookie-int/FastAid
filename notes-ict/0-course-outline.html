Course Description

This course provides a comprehensive introduction to computer systems, software, hardware, and architectures. Students will learn:

  • History and evolution of computers
  • Fundamentals of Information Systems
  • Number systems and arithmetic
  • Applications software: document processing, spreadsheets, presentations, databases
  • System software and development
  • Web and internet services
  • Introduction to operating systems and computer networks

Course Learning Outcomes (CLOs)

  • Understand computer systems, hardware, and software technologies
  • Learn basic logic, algorithms, and number system arithmetic
  • Gain practical knowledge of applications software
  • Learn basic web development using HTML, CSS, and JavaScript
  • Understand operating systems and network communication technologies
