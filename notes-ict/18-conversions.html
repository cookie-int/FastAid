Number Conversions — Step-by-step Worked Examples

This page contains clear methods and two worked examples each for the most common conversions used in computing: decimal, binary, octal and hexadecimal. Copy the examples into your notes or practice them on paper.

1) Decimal ↔ Binary

Decimal → Binary (Repeated Division by 2)

Method: divide the decimal number by 2 repeatedly, record remainders (0/1). Binary is remainders read bottom → top.

Example A — Convert 43 (decimal) → binary 
43 ÷ 2 = 21 rem 1
21 ÷ 2 = 10 rem 1
10 ÷ 2 = 5  rem 0
5  ÷ 2 = 2  rem 1
2  ÷ 2 = 1  rem 0
1  ÷ 2 = 0  rem 1

Remainders (bottom→top): 1 0 1 0 1 1 → (101011)₂
Check: 1·2⁵ +0·2⁴+1·2³+0·2²+1·2¹+1·2⁰ = 32+0+8+0+2+1 = 43
Example B — Convert 156 (decimal) → binary 
156 ÷ 2 = 78  rem 0
78  ÷ 2 = 39  rem 0
39  ÷ 2 = 19  rem 1
19  ÷ 2 = 9   rem 1
9   ÷ 2 = 4   rem 1
4   ÷ 2 = 2   rem 0
2   ÷ 2 = 1   rem 0
1   ÷ 2 = 0   rem 1

Remainders (bottom→top): 1 0 0 1 1 1 0 0 → (10011100)₂
Check: 128+0+0+8+4+2+0+0 = 156

Binary → Decimal (Positional weights)

Example C — Convert (101011)₂ → decimal 
Bits (from right, positions 0..):
1·2⁵ + 0·2⁴ + 1·2³ + 0·2² + 1·2¹ + 1·2⁰
= 32 + 0 + 8 + 0 + 2 + 1 = 43
Example D — Convert (10011100)₂ → decimal 
Positions: from right (0): bits 0..7
1·2⁷ + 0·2⁶ + 0·2⁵ + 1·2⁴ + 1·2³ + 1·2² + 0·2¹ + 0·2⁰
= 128 + 0 + 0 + 16 + 8 + 4 + 0 + 0 = 156

2) Decimal ↔ Hexadecimal

Decimal → Hexadecimal (Repeated Division by 16)

Example A — Convert 2748₁₀ → hexadecimal 
2748 ÷ 16 = 171 rem 12  → rem 12 = C
171  ÷ 16 = 10  rem 11  → rem 11 = B
10   ÷ 16 = 0   rem 10  → rem 10 = A

Read remainders bottom→top: A B C → (ABC)₁₆
Check: A·16² + B·16¹ + C·16⁰ = 10·256 + 11·16 + 12·1 = 2560 + 176 + 12 = 2748
Example B — Convert 605₁₀ → hexadecimal 
605 ÷ 16 = 37 rem 13  → D
37  ÷ 16 = 2  rem 5   → 5
2   ÷ 16 = 0  rem 2   → 2

Read bottom→top: 2 5 D → (25D)₁₆
Check: 2·256 + 5·16 + 13·1 = 512 + 80 + 13 = 605

Hexadecimal → Decimal (Positional)

Example C — Convert (ABC)₁₆ → decimal 
A·16² + B·16¹ + C·16⁰ = 10·256 + 11·16 + 12·1 = 2560 + 176 + 12 = 2748
Example D — Convert (25D)₁₆ → decimal 
2·16² + 5·16¹ + D·16⁰ = 2·256 + 5·16 + 13·1 = 512 + 80 + 13 = 605

3) Decimal ↔ Octal

Decimal → Octal (Repeated Division by 8)

Example A — Convert 724₁₀ → octal 
724 ÷ 8 = 90 rem 4
90  ÷ 8 = 11 rem 2
11  ÷ 8 = 1  rem 3
1   ÷ 8 = 0  rem 1

Read bottom→top: 1 3 2 4 → (1324)₈
Check: 1·8³ + 3·8² + 2·8¹ + 4·8⁰ = 512 + 192 + 16 + 4 = 724
Example B — Convert 87₁₀ → octal 
87 ÷ 8 = 10 rem 7
10 ÷ 8 = 1  rem 2
1  ÷ 8 = 0  rem 1

Bottom→top: 1 2 7 → (127)₈
Check: 1·64 + 2·8 + 7·1 = 64 + 16 + 7 = 87

Octal → Decimal

Example C — Convert (1324)₈ → decimal 
1·8³ + 3·8² + 2·8¹ + 4·8⁰ = 512 + 192 + 16 + 4 = 724
Example D — Convert (127)₈ → decimal 
1·8² + 2·8¹ + 7·8⁰ = 64 + 16 + 7 = 87

4) Binary ↔ Hexadecimal (use groups of 4)

Method: group binary bits into 4-bit chunks (from right). Convert each 4-bit group to 1 hex digit.

Example A — Convert (1010111011)₂ → hexadecimal 
Step 1: Group into 4s from right:  10 1011 1011  → pad left to make 4: 0010 1011 1011
Convert groups:
0010 → 2
1011 → B
1011 → B

Result: 2BB₁₆
Check (optional): 2·16² + B·16¹ + B·16⁰ = 2·256 + 11·16 + 11 = 512 + 176 + 11 = 699
Binary to decimal check: (1010111011)₂ = 699₁₀
Example B — Convert (110100111010)₂ → hex 
Group: 1101 0011 1010
1101 → D
0011 → 3
1010 → A

Result: D3A₁₆

Hexadecimal → Binary (each hex → 4 bits)

Example C — Convert (2BB)₁₆ → binary 
2 → 0010
B → 1011
B → 1011
Concatenate: 0010 1011 1011 → drop leading zeros if not needed: 1010111011₂
Example D — Convert (D3A)₁₆ → binary 
D → 1101
3 → 0011
A → 1010
Result: 110100111010₂

5) Binary ↔ Octal (use groups of 3)

Method: group binary bits into 3-bit chunks (from right). Each 3-bit group maps to an octal digit (0–7).

Example A — Convert (1011010111)₂ → octal 
Group from right in 3s: 1 011 010 111 → pad left: 001 011 010 111
001 → 1
011 → 3
010 → 2
111 → 7

Result: 1327₈
Check: Convert 1327₈ → binary groups → same result.
Example B — Convert (11101001)₂ → octal 
Group: 11 101 001 → pad left: 011 101 001
011 → 3
101 → 5
001 → 1

Result: 351₈

Octal → Binary (each octal digit → 3 bits)

Example C — Convert 1327₈ → binary 
1 → 001
3 → 011
2 → 010
7 → 111
Concatenate: 001011010111 → remove leading zeros if wanted: 1011010111₂
Example D — Convert 351₈ → binary 
3 → 011
5 → 101
1 → 001
Concatenate: 011101001 → leading zero optional → 11101001₂

6) Hexadecimal ↔ Octal

There is no direct single-step rule commonly taught — convert via binary (hex ↔ binary ↔ octal) because hex groups 4 bits and octal groups 3 bits.

Example A — Convert (2F3)₁₆ → octal 
Step 1: Hex → binary (each hex → 4 bits)
2 → 0010
F → 1111
3 → 0011
Binary: 0010 1111 0011 → combine: 001011110011

Step 2: Group into 3s from right: 00 101 111 001 1 → pad left: 000 101 111 001 1? better re-pad: 001 011 110 011
Actually correct grouping: 001 011 110 011

001 → 1
011 → 3
110 → 6
011 → 3

Result: 1363₈

(Always pad binary at left so its length is multiple of 3 for octal grouping.)
Example B — Convert (A4)₁₆ → octal 
Hex → binary:
A → 1010
4 → 0100
Binary: 1010 0100 → 10100100

Group into 3s from right: 10 100 100 → pad left: 010 100 100
010 → 2
100 → 4
100 → 4

Result: 244₈
Check (optional): 244₈ → decimal = 2·64 + 4·8 + 4 = 128 + 32 + 4 = 164
Hex A4 → decimal = 10·16 + 4 = 160 + 4 = 164 → matches

Final Notes & Tips

  • When grouping binary for hex use groups of 4; for octal use groups of 3. Always pad on the left with zeros if needed.
  • Use repeated division for decimal→base conversions and positional expansion for base→decimal conversions.
  • For fractional numbers: use multiplication (for decimal→other bases fractional part) and negative powers (for other bases→decimal fractional part).
  • Practice: convert numbers both ways until methods become routine. Check by converting back to decimal.
